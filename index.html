HTML,CSS and JS

HTML

HTML stands for Hyperlink Text Markup Language, and is commonly used for structuring and developing web pages. It is used to define the structure of a web page and it's content, and is often assisted by other tools such as CSS and Javascript.

  • A HTML tag is used to mark the start and end of a HTML element, it allows for formatting of the website and ordering elements.
  • HTML tags must be opened and closed to show where the start and end of the element is.
  • HTML attributes are tags used at the start and end of an element to change the element's behavior.
  • A HTML element is everything from the start to the end of a tag
